## About
A learning project built to explore the Internet Computer Protocol (ICP) blockchain. Deploys a simple Motoko canister with a JavaScript frontend that interacts through the Candid interface.

## What I Learned
- ICP actors and asynchronous query/update calls
- Motoko programming language syntax and features
- dfx CLI for local development and mainnet deployment
- Candid interface generation and cross-canister communication
- Webpack-based frontend integration with the Internet Computer

## Tech Stack
| Component | Technology |
|-----------|-----------|
| Backend | Motoko (Internet Computer) |
| Frontend | JavaScript, HTML, CSS |
| Toolchain | dfx SDK, Webpack |
| Platform | Internet Computer Protocol |

## Quick Start
1. Install dfx SDK
2. Start local environment: `dfx start --background`
3. Deploy canisters: `dfx deploy`
